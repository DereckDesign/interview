What is a website?

I see a website like an information book for a computer. It’s a digital place where people can read about the given information and can bookmark/save any website they want.

Can design change the world

I dont think design can change the world, however, i think design can change the world for someone. It can change the way people see the world and make them see diffrent things that they would’nt be able to see if it wasn’t for the designer’s work/art.
